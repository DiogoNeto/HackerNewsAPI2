HACKER NEWS API 2 - .NET6 ASP.NET WEB API

developed in Microsoft Visual Studio Community 2022 (64-bit) 

Open the solutions in file "Hacker News API 2.sln" and run it!

EndPoints:

Use swagger to do the requests:
GET: /api/News    (get all news ids)
GET: /api/News/{id}  (get the info of a news by id)
GET: /best20		(get the best 20 news)


or use a web browser:
https://localhost:7219/api/News		(get all news ids)
https://localhost:7219/api/News/{id}		(get the info of a news by id)
https://localhost:7219/best20				(get the best 20 news)
